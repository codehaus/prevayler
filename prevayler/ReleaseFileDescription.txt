Name
========

prevaylerX.Y.zip or prevaylerX.Y.Z.zip

Example: prevayler2.1.zip, prevayler2.3.1.zip, prevayler3.0.zip

The first and second numbers are determined by progress through a feature roadmap.
The third number, when present, indicates a bugfix patch release.

Directory Contents
======================

/prevaylerX.Y - All files in the project directory named "root" and its subdirectories.

/prevaylerX.Y/prevaylerX.Y.jar - .class files compiled from "src/main" except the demos. Demos are available in source code only.

/prevaylerX.Y/src - All files from "src/main" and its subdirectories.

/prevaylerX.Y/docs/api/beginners - Javadoc generated for package org.prevayler in "src/main" only.

/prevaylerX.Y/docs/api/advanced - Javadoc generated for the following packages in "src/main":
	org.prevayler
	org.prevayler.implementation
	org.prevayler.implementation.(all subpackages)

/prevaylerX.Y/lib - Required jars. Ex: The Skaringa jar.
