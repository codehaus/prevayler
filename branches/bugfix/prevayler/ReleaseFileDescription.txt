Name
========

prevaylerX.XX.XXXalpha.zip

Example: prevayler1.00.003alpha.zip

First number - incremented when a major functional or architectural goal is reached.

Second number - incremented when a functional or architectural goal is reached; or when backward compatibility is broken.

Third number - incremented when backward compatibility is maintained. This means all 1.02.XXX builds, for example, are backward compatible with 1.02.000.

Builds are generated with the "beta" suffix and have the suffix removed as they reach production stage.


Directory Contents
======================

/prevaylerX.XX.XXX - All files in the project directory named "root" and its subdirectories.

/prevaylerX.XX.XXX/prevaylerX.XX.XXX.jar - .class files compiled from "src/main" except the demos. Demos are available in source code only.

/prevaylerX.XX.XXX/src - All files from "src/main" and its subdirectories.

/prevaylerX.XX.XXX/docs/api/beginners - Javadoc generated for package org.prevayler in "src/main" only.

/prevaylerX.XX.XXX/docs/api/advanced - Javadoc generated for the following packages in "src/main":
	org.prevayler
	org.prevayler.implementation
	org.prevayler.implementation.(all subpackages)

/prevaylerX.XX.XXX/lib - Required jars. Ex: The Skaringa jar.
